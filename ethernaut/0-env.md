# 环境准备
 
这是我的第一次关于wargame的记录，我想还是要记录一下我是如何准备我的环境的。
目前感觉大概是需要：

- Chrome
- MetaMask

最开始的时候，我认为我是不需要使用 [harhat](https://hardhat.org/)  活着 [foundry](https://github.com/foundry-rs/foundry)，所以我先用浏览器就好了。
